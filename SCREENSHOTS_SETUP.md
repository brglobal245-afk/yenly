# App Screenshots Setup Guide

## Image Files Needed

You need to add **4 image files** to your project folder with these exact names:

1. **`screenshot-home.jpg`** - Home Dashboard (ホームダッシュボード)
   - The home screen with greeting, monthly summary, points balance, and upcoming payments

2. **`screenshot-expenses.jpg`** - Expense Management (支出管理)
   - The expense tracking screen showing categories and expense history

3. **`screenshot-points.jpg`** - Points Management (ポイント管理)
   - The points management screen showing all point cards (Rakuten, PayPay, etc.)

4. **`screenshot-income.jpg`** - Income Management (収入管理)
   - The income management screen showing income sources and totals

## How to Add the Images

### Option 1: Using File Explorer
1. Save your 4 screenshot images
2. Rename them to match the names above exactly:
   - `screenshot-home.jpg`
   - `screenshot-expenses.jpg`
   - `screenshot-points.jpg`
   - `screenshot-income.jpg`
3. Copy them into your project folder: `C:\Users\moksh\OneDrive\Desktop\yenly site`
4. Upload them to GitHub along with your other files

### Option 2: Using GitHub Website
1. Go to your repository on GitHub
2. Click **"Add file"** → **"Upload files"**
3. Drag and drop your 4 screenshot images
4. Make sure to rename them to the exact names listed above
5. Click **"Commit changes"**

## Image Requirements

- **Format**: JPG or PNG (both work)
- **Recommended size**: 1080x1920 pixels (9:16 aspect ratio for mobile screenshots)
- **File size**: Keep under 500KB each for fast loading
- **Optimization**: Use tools like TinyPNG or Squoosh to compress if needed

## After Adding Images

1. Refresh your website
2. The screenshots section will automatically display your images
3. Images will have hover effects and labels at the bottom
